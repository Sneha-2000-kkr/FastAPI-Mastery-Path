# Module 04 — Project Structuring (Layered/Clean Architecture)

A miniature **multi-resource** API (`/orders`, `/customers`) that demonstrates
how to organise a real backend: settings, lifespan, routers, services,
repositories, domain models, schemas, exceptions, and a unit-of-work pattern.

## Run
```bash
uvicorn app.main:app --reload --port 8000
```
